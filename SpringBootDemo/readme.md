What is this project about
=========
This is just a hanlp demo. 

How to build
============
1. install jdk1.8 x64 and maven

2. download hanlp data

4. install jdk1.8 x64 and set JAVA_HOME

5. use ide like eclipse/idea to open the project

6. debug or run
